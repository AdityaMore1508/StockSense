# 📈 StockSense — EMA Stock Trend Predictor


## About
StockSense is a web-based stock trend prediction dashboard for Indian retail investors. It uses **Exponential Moving Average (EMA-12 / EMA-26)** crossover analysis to predict whether a stock is heading into an **UPTREND** or **DOWNTREND** — helping users make data-driven decisions without emotional bias.

## Features
- 📊 EMA-12 / EMA-26 Golden Cross & Death Cross detection
- 🇮🇳 12+ NSE-listed Indian stocks (RELIANCE, TCS, INFY, HDFC & more)
- 🔮 Day-wise price forecast up to 30 days
- 📈 Interactive Chart.js visualizations (historical + predicted trends)
- 📱 Fully responsive — mobile, tablet & desktop ready

## Tech Stack
| Layer | Technology |
|---|---|
| Frontend | HTML5, CSS3, JavaScript |
| Charts | Chart.js 4.4.1 |
| Algorithm | EMA Crossover (EMA-12 / EMA-26) |
| UI Design | Figma |

## How It Works
1. Select any NSE-listed stock
2. System computes EMA-12 (fast) and EMA-26 (slow)
3. Crossover detected → signal generated
4. Predicted prices displayed for next 7–30 days
